# Grid Library Replacement

The project originally bundled Handsontable to provide spreadsheet-like tables. Because of licensing concerns, it has been replaced with our own open source grid library **VueGrid**.

## Features of VueGrid

- Dynamic column configuration
- Row drag-and-drop
- Inline cell editing
- Data import and export

## Difference from Handsontable

Handsontable offers a full spreadsheet experience, including formula support and complex header structures. VueGrid focuses on lightweight tables, so it does not implement formulas or nested headers. The library is smaller and fully open source, which makes it easier to integrate in this project.
